# IEEE754Inspector
 IEEE754检视工具  
 目前数据类型支持单精度与双精度.  
 功能有检视与编辑浮点数的二进制位, 以及查看并应用当前浮点数的最小增减量(Unit in Last Place)  
 使用了WPF UI
